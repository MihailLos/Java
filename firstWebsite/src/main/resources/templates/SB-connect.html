Инструкция по установке Spring Boot

1. Переходим на сайт, который сформирует начальный проект:

2. Выбираем тип создаваемого проекта на ваш вкус: Maven или Gradle (советую выбирать Maven)

3. Выбираем язык программирования, при помощи которого будет производиться работа с сайтом (в нашем случае Java)

4. Выбираем версию Spring Boot. Можно выбрать любую версию (лучше самую последнюю), главное не SNAPSHOT

5.1. Вводим имя группы (Group) в следующем формате: com.*name*: название группы не иметт значения

5.2. Вводим желаемое название проекта (Artifact)

5.3. Все остальные поля должны заполниться автоматически, их можно не трогать

6. Выбираем расширение создаваемого пакета (Packaging). Поскольку мы работаем с Java, выбираем расширение Jar

7. Выбираем версию Java (Java). Для этого вспомните, какую версию JDK вы ставили на компьютер, и выберите нужную

8. Нажимаем кнопку Generate, распаковываем полученный проект в нужное место и открываем его в предпочитаемой IDE

9. По итогу, развернутый проект должен выглядеть примерно так:

10. Теперь развернем сервер - для этого перейдите на сайт со следующим Гайдом

11. Следуем этому гайду и установка и развертывание Spring Boot сервера завершены
